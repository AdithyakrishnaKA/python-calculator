# Python Calculator

A beginner-friendly calculator project built using Python.

## About

This project performs basic arithmetic operations through user input.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by zero handling
* Invalid operator handling

## Concepts Learned

* Variables
* User input
* Conditional statements (`if`, `elif`, `else`)
* Type conversion
* Basic error handling

## How to Run

```bash
python calculator.py
```
